# Security Policy

## Supported Versions
This project is pre-alpha. Security fixes are applied to `main` only.

## Reporting a Vulnerability
Please do not open public issues for vulnerabilities.

Report privately with:
- Description of the issue
- Reproduction steps or proof of concept
- Affected components and impact assessment

Until a dedicated address is added, open a private GitHub security advisory in the repository.

## Response Targets
- Initial acknowledgment: within 72 hours
- Triage decision: within 7 days
- Fix timeline: based on severity and complexity
